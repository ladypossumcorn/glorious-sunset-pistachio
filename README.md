# Glorious Sunset
Gruvbox-based theme for [Revenge](https://github.com/revenge-mod/revenge-bundle) with edited accents. This is my first attempt at theming.

# Installation
Go to **Themes** under **Revenge** in **Settings**, then tap the (+) button, finally paste this URL:

```
https://github.com/eeriemyxi/glorious-sunset/raw/refs/heads/main/theme.json
```
